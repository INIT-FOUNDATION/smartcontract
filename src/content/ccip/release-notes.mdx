## v1.2.0 release on testnet - 2023-12-08

### Updates

- Added support for USDC transfers. For applicable lanes, refer to the [Supported Networks](/ccip/supported-networks/v1_2_0/testnet).

- Increased gas limit for CCIP messages. See the updated [Service Limits](/ccip/service-limits).

### Notices

- CCIP v1.0.0 is now deprecated on testnet. Migration to v1.2.0 is required **by January 31st, 2024**. Configuration details (including the new router addresses) are available on the [CCIP v1.2.0 Configuration](/ccip/supported-networks/v1_2_0/testnet) page.

## Arbitrum Goerli - 2023-11-17
